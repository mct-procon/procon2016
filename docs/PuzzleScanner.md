# PuzzleScannerの使い方  
## 画面1 "Choose Acceralator"  
"OpenCL"と"None(CPU)"とボタンがありますが，自分のPCに自信のある方はOpenCLのほうを選んでください．  
本番のパソコンでは"OpenCL"を選んでください．  
## 画面2 "Choose Image"  
"Reference"にて，画像を参照して，Previewでその画像が正しいかを確認したうえで，"Next"ボタンを押してください．  
## 画面3 "Choose HSV Filtering"  
HSVでフィルタリングします．  
上のHSVのスライダーによりそれぞれの値の最大値・最小値を設定します．  
上のスライダーは最大値，下のスライダーは最小値になっています．  
変更したら，右下のUpdateボタンを押して，画面を更新してください．  
## 画面4  
枠かどうかの選択：  
 * 左からFrameボタンを押してトグルする．  
 * 辺(緑色)を右クリックする．  

誤認識のものかどうかの選択：  
 * 左からトグルボタンを押してトグルする．  
 * 辺(緑色)を左クリックする．  

"Shot"ボタン：スクリーンショットします．  
"IsShowSize"トグルボタン：??px や ??°の表示を消す．
